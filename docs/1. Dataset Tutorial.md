## Dataset Setup

### Preprocessed Path

    unzip All_Sources_preprocessed.zip -d /content/data/preprocessed/

### Raw Path

    unzip <dataset>.zip -d /content/data/raw/

------------------------------------------------------------------------

## Data Format

-   Inputs: NumPy arrays (H, W) or (H, W, 3)
-   Labels: real, ai

------------------------------------------------------------------------

## Extending Dataset

    [label]_[dataset]_[index].png

## Dataset Splits

    metadata/splits/train_metadata.csv
    metadata/splits/test_metadata.csv

------------------------------------------------------------------------

## Summary

-   GitHub contains code, notebooks, and metadata
-   Google Drive stores datasets as ZIP archives
-   Data is extracted locally in Colab
-   Preprocessed dataset is the default path
-   Raw datasets are optional for learning
