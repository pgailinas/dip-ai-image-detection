# 7. Results & Insights

## Overview

This section presents the experimental results and analytical insights obtained from the Digital Image Processing (DIP)–based AI image detection pipeline. The objective is not only to report classification performance, but also to examine how different feature types, model choices, and dataset combinations influence overall detection behavior and generalization capability.

The experiments evaluate the effectiveness of a handcrafted 26-feature DIP representation derived from gradient-based, spatial-domain, and frequency-domain image characteristics. Multiple classifiers, feature subsets, and dataset pairings were analyzed to better understand which components contribute most strongly to robust AI-image detection.

The results are organized into the following major analysis categories:

* **Classifier Comparison** — Evaluation of multiple baseline classifiers using stratified cross-validation and independent test metrics
* **Final Model Selection** — Summary of the selected tuned model using standard evaluation metrics and ROC analysis
* **Feature-Level Results** — Analysis of the discriminative strength of individual DIP features
* **Group-Level Results** — Comparison of gradient, spatial, and frequency-domain feature groups
* **Source Pair Analysis** — Evaluation of model robustness across multiple real-versus-AI dataset combinations
* **Generalization Insights** — Interpretation of cross-source behavior and generator-independent detection performance

Together, these results demonstrate how explicit DIP feature engineering can provide strong classification performance while also offering interpretability and improved insight into the statistical differences between real and AI-generated imagery.

---

## Classifier Comparison

This experiment evaluated multiple baseline classifiers using the normalized 26-feature DIP representation. All models were trained and evaluated using stratified 5-fold cross-validation, with mean ROC-AUC used as the primary ranking metric.

The objective of this stage was to determine which classifier architectures were best suited for separating real and AI-generated images using handcrafted DIP features before performing final model tuning and independent testing.

<p align="center">
  <img src="images/classifier-comparison.png" width="700">
</p>
<p align="center"><em>Figure: Baseline classifier comparison ranked by mean ROC-AUC</em></p>

The results show that nonlinear classifiers consistently outperformed simpler linear and probabilistic approaches. The top-performing models were:

| Rank | Classifier | Mean ROC-AUC |
|---|---|---:|
| 1 | RBF SVM | 0.8608 |
| 2 | MLP | 0.8594 |
| 3 | Random Forest | 0.8465 |
| 4 | Extra Trees | 0.8440 |
| 5 | Gradient Boosting | 0.8312 |

Several important observations can be made from these results:

* **RBF SVM** achieved the highest overall ROC-AUC, suggesting that nonlinear decision boundaries are highly effective for modeling the statistical separation between real and AI-generated imagery within the DIP feature space.
* **MLP** produced nearly identical performance to the RBF SVM, indicating that the DIP feature representation is also highly compatible with neural-network-based classification.
* **Tree ensemble methods** (Random Forest, Extra Trees, and Gradient Boosting) demonstrated strong performance, implying that many DIP features contain useful nonlinear interactions and hierarchical relationships.
* **Linear models** such as Logistic Regression and Linear SVM performed noticeably worse, suggesting that the separation between real and synthetic imagery is not purely linearly separable.
* **Gaussian Naive Bayes** produced the weakest performance, likely because the DIP features violate the model’s independence assumptions.

Although the RBF SVM achieved the highest untuned ROC-AUC, both the RBF SVM and MLP demonstrated strong and stable performance and were therefore selected for subsequent tuning and final evaluation experiments.

Overall, the classifier comparison results indicate that the handcrafted DIP feature representation contains meaningful discriminative information that can be effectively leveraged by multiple nonlinear learning algorithms.

---

## Final Model Selection

Following baseline classifier comparison and hyperparameter tuning, the final model evaluation stage focused on the two strongest candidate classifiers:

* **RBF SVM**
* **MLP (Multi-Layer Perceptron)**

Both models were trained using the normalized 26-feature DIP representation and evaluated on an independent test dataset that was completely isolated from training and cross-validation procedures. This separation ensured that final performance metrics reflected true generalization behavior rather than optimization bias or data leakage.

The tuned MLP model ultimately achieved the strongest overall balance across evaluation metrics and was therefore selected as the primary model for subsequent feature analysis and source-pair experiments.

Final evaluation metrics included:

* **Accuracy** — overall classification correctness
* **Precision** — reliability of AI-generated image predictions
* **Recall** — ability to correctly detect AI-generated images
* **F1-score** — balanced measure of precision and recall
* **ROC-AUC** — threshold-independent measure of class separability

The final comparison between the tuned RBF SVM and tuned MLP models demonstrated that both classifiers generalized effectively on unseen data, confirming that the handcrafted DIP feature representation captures meaningful statistical differences between real and synthetic imagery.

The MLP classifier was ultimately favored because of its slightly stronger overall performance and its ability to model complex nonlinear relationships among gradient, spatial, and frequency-domain features.

## Feature-Level Results

Feature-level analysis examined the predictive power of each of the **26 DIP features** individually.

This analysis revealed:

* Certain features provide strong standalone predictive capability
* Other features contribute primarily in combination with others
* The combination of **gradient**, **spatial**, and **frequency-domain** features improves overall model performance

These results reinforce the importance of **multi-domain feature representation** when distinguishing real and AI-generated images.

## Group-Level Results

Group-level analysis examined the performance contribution of the three major DIP feature groups: **gradient-based**, **spatial-domain**, and **frequency-domain** features.

This analysis revealed:

* Each feature group captures a different type of image evidence
* Some groups provide stronger standalone separation between real and AI-generated images
* Combining feature groups provides a more complete representation than relying on any single group alone

These results support the use of a multi-domain DIP feature vector for AI-generated image detection.

## Source-Pair Analysis

To evaluate generalization, a total of **9 source-pair experiments (3 real × 3 AI)** were conducted.

Each experiment:

* Trained a model using a specific real dataset and AI dataset pairing
* Evaluated performance using a balanced test set
* Compared results using ROC AUC and other metrics

This analysis provides insight into how model performance varies depending on dataset composition.

## Generalization Insights

The source-pair analysis highlights an important characteristic of the model:

> **Performance varies across dataset combinations.**

Key observations include:

* Some dataset pairs exhibit strong separability between real and AI images
* Other combinations are more challenging, indicating overlap in feature distributions
* Differences suggest that **dataset characteristics and generation methods influence detectability**

These findings emphasize that:

* High overall accuracy does not guarantee consistent performance across all data sources
* Evaluating generalization is critical when developing AI detection systems

---

## Key Takeaways

* DIP-based feature engineering provides strong classification performance
* Combining multiple feature domains improves robustness
* Model performance is influenced by dataset selection and characteristics
* Source-pair analysis is essential for understanding generalization behavior
* ROC curve analysis provides deeper insight than single-value metrics alone
